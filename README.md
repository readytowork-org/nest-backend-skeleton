# NestJS Skeleton

A production-ready NestJS application template with best practices and common utilities.

## 🚀 Quick Start

```bash
# Install dependencies
yarn install

# Setup environment
cp .env.example .env

# Start development server
yarn start:dev
```

## 🛠️ Features

- NestJS with TypeScript
- PostgreSQL + Drizzle ORM
- JWT Authentication
- Swagger API docs
- Environment config
- Logging system
- Jest testing
- ESLint + Prettier
